需要加入Apostrophe文档和页面的生命周期吗？Apostrophe在保存文档、呈现页面等时发出"promise events"。[阅读"promise events" 参考]()

## Numjucks 过滤器

Apostrophe为Numjucks模板语言提供了支持，它与Jinja和Twig兼容。除了许多富足函数外，Apostrophe还提供了额外的Numjucks"过滤器(filters)"来转换输出数据。[阅读Numjucks 引用]()
